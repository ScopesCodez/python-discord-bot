# Requirements
- Python 3.7 or higher.
- disnake library.
- discord.py library installed from disnake.

# Setup
Clone this repository
```
git clone https://github.com/ScopesCodez/python-discord-bot/
```
Install the required packages
```
pip3 install -r requirements.txt
```
